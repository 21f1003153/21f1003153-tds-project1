Simple Quiz App

What is 2 + 2?

What color do you get by mixing red and white?

Which animal is known as the King of the Jungle?

This HTML file contains all the elements needed for the interactive quiz. Each question has a set of radio buttons as options. Once the submit button is clicked, the JavaScript function `submitAnswers()` checks all the answers, calculates the score, and displays it.
